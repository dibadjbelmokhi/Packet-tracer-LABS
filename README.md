# Network Security & Cisco Labs

This repository contains a collection of networking and security labs completed during my Master's studies in Networks and Distributed Systems (RSD).

The projects focus on practical implementation of network technologies and security concepts using Cisco Packet Tracer and GNS3.

## Topics Covered

- VLAN Configuration
- Inter-VLAN Routing
- Static Routing
- Dynamic Routing
- Access Control Lists (ACLs)
- DMZ (Demilitarized Zone) Implementation
- Firewall Configuration
- Network Segmentation
- Switch and Router Configuration
- Network Security Fundamentals
- Troubleshooting and Connectivity Testing

## Tools

- Cisco Packet Tracer
- Cisco IOS

## Objectives

These labs were designed to develop hands-on skills in network design, configuration, security, and troubleshooting in enterprise-like environments.
